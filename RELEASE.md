# 版本发布记录

## v1.0.0 (2026-06-19)

### 🎉 首次发布

局域网聊天室 v1.0 正式发布！这是一个功能完整的局域网实时聊天应用。

### ✨ 核心功能

#### 实时聊天
- 基于 WebSocket 的实时消息通信
- 支持多人同时在线聊天
- 消息实时同步推送到所有客户端

#### 多媒体支持
- **图片发送**：支持发送 JPG、PNG、GIF 等格式图片
- **截图发送**：按 `Ctrl+V` 直接粘贴截图发送
- **文件传输**：支持发送任意格式文件，最大 32MB
- **图片预览**：点击图片可全屏预览

#### 用户体验
- **表情选择器**：内置 64 个常用表情
- **在线用户列表**：实时显示当前在线用户
- **扫码加入**：点击 ⚙️ 按钮显示二维码，手机扫码即可加入
- **响应式设计**：完美适配手机、平板、电脑

### 📦 技术栈

| 组件 | 技术 |
|------|------|
| 后端语言 | Go 1.26+ |
| WebSocket | gorilla/websocket |
| 二维码 | qrcodejs (CDN) |
| 前端 | 原生 HTML/CSS/JS |

### 🚀 快速开始

#### 编译
```bash
go build -o chat.exe .
```

#### 运行
```bash
./chat.exe
```

#### 访问
启动后根据终端提示访问：
- 本机：http://localhost:8080
- 局域网：http://<服务器IP>:8080

### 📋 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-port` | 服务器端口 | 8080 |
| `-static` | 静态文件目录 | static |
| `-upload` | 上传文件目录 | uploads |

### 📖 使用指南

1. **加入聊天**
   - 打开浏览器访问聊天室
   - 输入昵称，点击"加入聊天"

2. **发送消息**
   - 在输入框输入文字，按 Enter 或点击"发送"

3. **发送图片**
   - 点击 🖼️ 按钮选择图片
   - 或按 `Ctrl+V` 粘贴截图

4. **发送文件**
   - 点击 📎 按钮选择文件

5. **发送表情**
   - 点击 😀 按钮选择表情

6. **扫码加入**
   - 点击 ⚙️ 按钮显示二维码
   - 手机扫描二维码即可加入

7. **预览图片**
   - 点击聊天中的图片可全屏预览
